# Thread para ela

Projeto simples: uma página estática que mostra uma thread com 9 postagens — cada uma com uma foto e a última com um vídeo.

Como usar

-- Abra `index.html` no navegador (duplo-clique ou `File -> Open`).
-- Coloque suas imagens dentro da pasta `img/` no projeto com os nomes `1.jpg` a `8.jpg`.
-- Coloque a imagem de perfil nomeada `profile.jpg` dentro de `img/`.
-- O vídeo atualmente usa um sample público; para usar um vídeo local, substitua a `source` em `index.html` por `img/seu_video.mp4` (ou outro nome que preferir).

Arquivos criados

- `index.html`
- `styles.css`
- `script.js`
- `README.md`
